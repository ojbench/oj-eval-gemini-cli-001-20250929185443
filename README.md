# Problem 001 - A+B Problem

**ACMOJ Problem ID**: 1000

## 题目描述

输入两个整数，输出它们的和。

## 输入格式

一行，两个空格隔开的整数 A，B。

## 输出格式

一个数 A + B。

## 样例

### 输入
```
1 1
```

### 输出
```
2
```

- **内存空间限制**：256 MiB
